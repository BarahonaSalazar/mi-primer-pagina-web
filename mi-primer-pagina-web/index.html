Introducción a HTML

El World Wide Web (WWW) es un sistema que contiene una cantidad de información casi infinita. Pero, esta información debe estar ordenada de alguna forma de manera que sea posible encontrar lo que se busca. La unidad básica donde está almacenada esta información son las páginas Web.

Una página web es un documento digital que puede contener texto, sonido, video, programas, enlaces, imágenes, hipervínculos, entre muchas otras cosas. Se puede acceder a ellas desde un navegador web como Chrome, Firefox, Edge y una conexión activa a internet.

Las páginas web se encuentran programadas en formato HTML o XHTML y se pueden conectar entre ellas a través de hipervínculos, que son enlaces hacia diversos contenidos. Frecuentemente también incluyen otros recursos como pueden ser hojas de estilo en cascada, scripts, imágenes digitales, entre otros.

PERO ¿QUÉ ES HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado de hipertexto que permite crear y estructurar secciones, párrafo encabezados y enlaces para páginas web y aplicaciones.

Básicamente, es un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web como imágenes, listas, entre otros muchos posibles. Es fácil de aprender y entendible para el lenguaje humano.

HISTORIA DE HTML

El origen de HTML se remonta a 1980 cuando Tim Berners-Lee y su equipo lo crearon. El primer documento formal fue presentado en 1991 bajo el nombre de “HTML Tags” (Etiquetas HTML).

En 1995 el organismo IETF público el estándar HTML 2.0 que es el primer estándar oficial de HTML (a pesar de su nombre 2.0).

En 1997 se publicó la versión HTML 3.2, la primera publicada por W3C.

HTML 4 se publicó en abril de 1998, y supuso un gran salto desde las versiones anteriores. Entre sus novedades más destacadas se encuentran las hojas de estilos CSS, la posibilidad de incluir pequeños programas o scripts en las páginas web, mejora de la accesibilidad de las páginas diseñadas, tablas complejas y mejoras en los formularios.

HTML 5 fue publicado en 2014 en su versión final y contiene nuevos elementos, atributos y comportamientos. Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance.

Entorno de desarrollo

Un entorno de desarrollo o IDE es un editor de texto con avanzadas herramientas para el desarrollo de aplicaciones.

Visual Studio Code es un editor de texto que nos permite trabajar con diversos lenguajes de programación. Es gratuito, de código abierto y nos proporciona una utilidad para descargar y gestionar extensiones con las que podemos personalizar y potenciar esta herramienta.

Podemos profundizar leyendo la documentación de su página oficial https://code.visualstudio.com/
